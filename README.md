# 🧠 Deep Learning Projects & Labs

Welcome to the **DEEP-LEARNING** repository! This repository contains a collection of Jupyter Notebooks demonstrating various Deep Learning concepts, architectures, and projects. 

## 📂 Repository Contents

Here is a breakdown of the uploaded notebooks and what they cover:

### 🖼️ Image Classification & CNNs
- **`Brain_Tumor.ipynb`**: A deep learning project focused on detecting and classifying brain tumors from medical image data.
- **`CIFAR_10_Aman.ipynb`** & **`cifar-10.ipynb`**: Implementations of image classification models trained on the popular CIFAR-10 dataset.
- **`MNIST_Aman.ipynb`**: Handwritten digit classification using the classic MNIST dataset.
- **`CNN_DataAug_Dropout_MNIST_MinPooling.ipynb`**: An advanced Convolutional Neural Network (CNN) applied to the MNIST dataset, featuring data augmentation, dropout regularization, and custom pooling strategies like min-pooling.

### ⚙️ Core Architectures & Foundations
- **`MLP_from_scratch_numpy_lab.ipynb`**: A foundational lab where a Multi-Layer Perceptron (MLP) neural network is built entirely from scratch using only the `NumPy` library—without the use of high-level DL frameworks.
- **`TRANSFORMER.ipynb`**: Implementations or experiments demonstrating the Transformer architecture, typically used for advanced NLP or sequence modeling tasks.

### 📚 General & Combined Labs
- **`DeepLearning.ipynb`**: General deep learning experiments, code snippets, or lab exercises.
- **`allcombine.ipynb`**: A consolidated notebook combining multiple deep learning models, concepts, or pipeline steps.

## 🛠️ Technologies Used
- **Language**: Python 3.x
- **Core Libraries**: NumPy, Pandas, Matplotlib
- **Environment**: Jupyter Notebook

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Xnirc/DEEP-LEARNING.git
   cd DEEP-LEARNING
   ```

2. **Install dependencies:**
   Ensure you have Python installed, then install standard deep learning and data science libraries (like `numpy`, `matplotlib`, `tensorflow` or `pytorch`, depending on the specific notebook):
   ```bash
   pip install jupyter numpy pandas matplotlib
   ```

3. **Launch Jupyter Notebook:**
   ```bash
   jupyter notebook
   ```
   *Navigate to the desired `.ipynb` file in the browser to view and run the code.*
